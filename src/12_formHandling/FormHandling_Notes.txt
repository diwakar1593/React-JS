Formik
=======
Library for form handling. You can use the Higher Order Component approach or Render Props approach or Hook approach.

1) Setting of initial values in a form.     // writing mapPropsToValues function
2) Event handling for the various form fields   // fields
3) Validation   // Yup (errors, touched)
4) Handling Submission of the form.

https://formik.org/doc/overview
